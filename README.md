# Emma in danger: Escape from the Forest 🎮

**Emma in danger** – это 2D-платформер с элементами экшена, написанный на Python с использованием библиотеки Pygame.
Помогите девочке Эмме сбежать из заколдованного леса, полного опасных призраков. Собирайте очки,
сражайтесь с врагами и найдите выход, чтобы победить!

## Особенности ✨
- Динамичный геймплей с бесконечным раннером и элементами платформера.
- Три уровня сложности: **Easy**, **Normal**, **Hard** (меняются в меню).
- Система здоровья и боеприпасов.
- Бонусы: красные (лечение) и синие (дополнительные патроны).
- Случайно генерируемые платформы, на которые можно запрыгивать.
- Анимированные персонажи: Эмма и монстры.
- Музыкальное сопровождение.

## Управление ⌨️
| Клавиша | Действие |
|---------|----------|
| ← →     | Движение влево/вправо |
| Пробел  | Прыжок |
| Z       | Выстрел (тратит патроны) |
| ESC     | Выход в меню (во время игры) |

В меню также можно настроить сложность и посмотреть управление.

## Установка и запуск 🚀
1. Убедитесь, что у вас установлен Python 3.8+ и pip.
2. Установите зависимости:
   pip install pygame

Клонируйте репозиторий:
git clone https://github.com/finxond/Game-in-PyGame.git

Перейдите в папку игры и запустите:
python main.py

Если при запуске отсутствуют графические файлы, игра автоматически создаст цветные заглушки в папке resources.

Структура проекта 📁
main.py – главный файл, запуск игры.

settings.py – все настройки, константы.

utils.py – вспомогательные функции (загрузка изображений, отрисовка текста и др.).

menu.py – класс меню.

classes/ – папка с классами спрайтов:

player.py – игрок (Эмма).

enemy.py – враги (монстры).

bullet.py – пули.

powerup.py – бонусы.

platform.py – платформы.

resources/ – папка с ресурсами (изображения, звуки, шрифты). Создаётся автоматически, если отсутствует.

Зависимости 📦
Python 3.8+

Pygame 2.0+
